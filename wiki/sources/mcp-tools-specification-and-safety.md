# Source: Model Context Protocol — Tools and Safety

- **Author:** Model Context Protocol project
- **Published:** 2025-03-26 specification; draft tools page
- **URL:** https://modelcontextprotocol.io/specification/2025-03-26/index
- **Retrieved:** 2026-07-11

## Summary

MCP defines tools as model-controlled functions with names and schemas, while warning that tools can
represent arbitrary code execution. Its security guidance calls for explicit user consent, clear
understanding of tool behavior, access controls, and caution around untrusted tool descriptions. The
tools documentation recommends a human in the loop who can deny invocations.
