# Source: Unity — Input System

- **Author:** Unity Technologies
- **Published:** Current Unity 6.0 manual
- **URL:** https://docs.unity3d.com/kr/current/Manual/Input.html
- **Retrieved:** 2026-07-11

## Summary

Unity recommends the Input System package for new input work and describes it as the flexible,
extensible path with an editor for configuring input. The legacy Input Manager remains available for
older projects or compatibility needs. The practical choice is therefore architectural: a migration
should account for existing polling and serialized bindings rather than simply changing an API call.
