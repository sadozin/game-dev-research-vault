## Summary

Epic states that one Unreal Unit equals one centimeter and recommends verifying imported dimensions.
Its FBX pipeline supports assets authored in applications including Blender and uses FBX 2020.2;
import options include uniform scaling and separate static or skeletal mesh paths. The pipeline's
version and import settings are therefore part of a repeatable asset contract.
