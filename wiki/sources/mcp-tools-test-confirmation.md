# Source: MCP — Tool Confirmation for Test Runs

- **Author:** Model Context Protocol project
- **Published:** 2025-06-18 specification
- **URL:** https://modelcontextprotocol.io/specification/2025-06-18/server/tools
- **Retrieved:** 2026-07-11

## Summary

MCP's tool interaction guidance calls for clear exposure, invocation indicators, and confirmation
prompts. Applied to a Unity test runner, the user should see the selected mode, test scope, timeout,
and whether the run can modify scenes or assets before approving it.
