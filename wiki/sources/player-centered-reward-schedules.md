# Source: Player-Centered Positive Reinforcement in a Serious Game

- **Author:** International Journal of Serious Games
- **Published:** 2014
- **URL:** https://journal.seriousgamessociety.org/index.php/IJSG/article/view/47
- **Retrieved:** 2026-07-11

## Summary

The study compares fixed-ratio and variable-ratio reward schedules in a game-like memory task and
examines enjoyment and performance. It provides evidence that schedule choice changes player experience
and should be evaluated as part of the game context, not assumed to have one universal effect.
