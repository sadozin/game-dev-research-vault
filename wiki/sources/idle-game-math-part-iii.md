## Summary

Prestige resets temporary run progress in exchange for persistent acceleration. It creates the
feeling of climbing further on each run and also turns unmanageably large production numbers into
a slower-moving meta-progression variable. The article distinguishes common fractional-exponent
prestige rewards from a true logarithmic reset: both slow growth, but they shape later-game scale
differently.

Offline earnings are part of progression pacing, not an incidental timer. A cap changes whether
the player is expected to return for a meaningful decision or simply wait for passive gains, so
it must be balanced against the normal progress wall and prestige cadence.
