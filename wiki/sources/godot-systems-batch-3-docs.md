## Summary

Web: WebGL2 Compatibility only; single-thread export default; Sample audio limits effects;
HTTPS/cookies for persistence; COOP/COEP for threads; no C# on Godot 4 web. Dedicated server:
export mode strip visuals; `dedicated_server` feature; `--headless`. Navmesh: bake from
geometry/collision; agent_radius margin; chunks with border_size; parse on main thread.
Debug: Remote tree, visible collision/nav, multi-instance run, breakpoints, profiler process.
Transforms: avoid Euler for gameplay; use Basis/−Z forward; orthonormalize; FPS angle
accumulators; slerp quaternions.
