# Source: Epic — Using Gameplay Abilities

- **Author:** Epic Games
- **Published:** Undated (Unreal Engine 5.7/5.8 documentation)
- **URL:** https://dev.epicgames.com/documentation/en-us/unreal-engine/using-gameplay-abilities-in-unreal-engine
- **Retrieved:** 2026-07-11

## Summary

Epic documents ability costs, cooldowns, input, animation montages, tags, and gameplay events as
ability concerns. Its replication guidance distinguishes replicating an ability's internal state
from replicating activation or ending, and describes local prediction as immediate client execution
with the server retaining final authority. The overview also recommends Gameplay Effects for
attribute, tag, and cue changes because their lifetime and rollback behavior fit the system.
