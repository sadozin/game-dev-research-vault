## Summary

An Animation Blueprint controls a skeletal mesh's final pose at runtime through specialized visual
graphs. It separates the asset's animation behavior from the mesh itself and supports blending,
bone control, and state logic. The character must explicitly use the Animation Blueprint for the
graph to affect it.
