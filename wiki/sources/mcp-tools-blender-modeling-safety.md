- **Author:** Model Context Protocol project
- **Published:** 2025-06-18 specification
- **URL:** https://modelcontextprotocol.io/specification/2025-06-18/server/tools

## Summary

MCP recommends clear tool visibility, invocation indicators, and confirmation prompts. A Blender
modeling server should therefore expose the target object, operation, parameters, and expected scope
before it mutates mesh data.
