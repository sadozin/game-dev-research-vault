# Source: MCP — Tools for Scene Authoring Safety

- **Author:** Model Context Protocol project
- **Published:** 2025-06-18 specification
- **URL:** https://modelcontextprotocol.io/specification/2025-06-18/server/tools
- **Retrieved:** 2026-07-11

## Summary

The MCP tools specification treats tools as model-controlled functions discovered through schemas and
calls. It recommends visible tool exposure, clear invocation indicators, and confirmation prompts so
a human can deny operations. Those requirements favor narrow, previewable scene mutations over a raw
editor execution endpoint.
