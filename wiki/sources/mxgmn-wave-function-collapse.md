# Source: Maxim Gumin Wave Function Collapse repository

- **Author:** Maxim Gumin
- **Published:** 2016
- **URL:** https://github.com/mxgmn/WaveFunctionCollapse
- **Retrieved:** 2026-07-11

## Summary

Wave Function Collapse represents cells as sets of possible patterns, repeatedly collapses a choice,
and propagates local adjacency constraints. It produces varied outputs from examples or rules, but
local consistency is not global solvability: contradictions can require restart, backtracking, or
carefully designed constraints. The algorithm is a constraint solver, not a guarantee of good level
design.
