# Source: Unity animation compression documentation

- **Author:** Unity Technologies
- **Published:** continuously maintained documentation
- **URL:** https://docs.unity3d.com/Manual/AnimationCompression.html
- **Retrieved:** 2026-07-11

## Summary

Unity’s animation import pipeline offers key reduction and compression choices that trade clip size
against visual fidelity. The right setting depends on the motion and rig: a small error in a root or
contact track is more visible than the same error in a secondary joint. Compression must be checked
in the assembled animation graph, not only in an inspector preview.
