# Source: MCP — Tool Schemas and Confirmation

- **Author:** Model Context Protocol project
- **Published:** 2025-06-18 specification
- **URL:** https://modelcontextprotocol.io/specification/2025-06-18/server/tools
- **Retrieved:** 2026-07-11

## Summary

MCP tools are discovered through schemas and invoked as model-controlled operations. The specification
encourages clear UI indicators and confirmation prompts. A material tool should therefore expose the
target renderer, asset scope, property, and resulting mutation before applying it.
