# Source: Unity — About Netcode for GameObjects

- **Author:** Unity Technologies

- **URL:** https://docs-multiplayer.unity3d.com/netcode/2.3.2/about/

## Summary

Unity describes Netcode for GameObjects as a high-level networking library that abstracts low-level
protocol work while sending GameObjects and world data across a session. The documentation lists
networking, components, objects, messaging, serialization, scenes, debugging, and multiplayer
architecture as separate concerns, and records supported Unity versions and platforms.
