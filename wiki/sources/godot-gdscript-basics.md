## Summary

GDScript is Godot's high-level, indentation-based, gradually typed language. Scripts act as
classes (`extends`, optional `class_name`). Core features: typed vars, enums, signals,
`@export` / `@onready`, `await`, vectors/colors as builtins, typed arrays/dictionaries.
Integer division truncates (`5 / 2 == 2`). Style: PascalCase types, snake_case funcs/vars.
Not Python — similar look only.
