# Source: procedural level generation and LLM limitations references

- **Author:** Amit Patel; Model Context Protocol maintainers
- **Published:** continuously maintained references
- **URL:** https://www.redblobgames.com/articles/level-design/
- **Retrieved:** 2026-07-11

## Summary

Level design needs spatial constraints, pacing, navigability, authored intent, and playtesting. An LLM
can propose structures or call tools, but it does not observe the resulting player experience unless the
workflow supplies measurements and checks. Generated output therefore needs deterministic validation,
human review, and an escape hatch for designs that satisfy syntax but fail as play spaces.
