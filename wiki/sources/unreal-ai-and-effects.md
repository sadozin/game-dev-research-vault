## Summary

Niagara separates emitters and systems for modular effects; Behavior Trees execute decision branches
using Blackboard state; AI Perception converts registered stimuli into sensory updates. Each system
provides useful authoring structure, but runtime cost, stale facts, and excessive graph coupling remain
design responsibilities.
