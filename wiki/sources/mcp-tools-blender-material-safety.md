# Source: MCP — Material Graph Tool Safety

- **Author:** Model Context Protocol project
- **Published:** 2025-06-18 specification
- **URL:** https://modelcontextprotocol.io/specification/2025-06-18/server/tools
- **Retrieved:** 2026-07-11

## Summary

MCP recommends visible tools, invocation indicators, and user confirmation. A material-node tool
should show the targeted material, engine, node/link changes, and affected objects before applying a
graph mutation.
