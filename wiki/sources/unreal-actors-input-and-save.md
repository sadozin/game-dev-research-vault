# Source: Unreal actors, Enhanced Input, and save-game documentation

- **Author:** Epic Games
- **Published:** continuously maintained documentation
- **URL:** https://dev.epicgames.com/documentation/en-us/unreal-engine/gameplay-framework-in-unreal-engine
- **Retrieved:** 2026-07-11

## Summary

Unreal separates world Actors from reusable Actor Components, routes input through mapping contexts
and actions, and provides SaveGame objects as an explicit persistence boundary. These systems are
powerful because responsibilities are named and composable; they still require lifecycle ownership,
authority, and versioning decisions from the game.
