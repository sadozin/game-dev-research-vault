## Summary

Epic describes Blueprint Classes as assets that combine components, variables, and node-based behavior.
The Blueprint Editor supports setup, event responses, modular operations, and debugging; Construction
Scripts execute when an actor is placed or updated in the editor. Blueprint changes apply to instances,
so authoring scope matters.
