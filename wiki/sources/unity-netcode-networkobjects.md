# Source: Unity — Netcode for GameObjects

- **Author:** Unity Technologies

- **URL:** https://docs.unity3d.com/ja/current/Manual/com.unity.netcode.gameobjects.html

## Summary

Unity positions Netcode for GameObjects as a high-level SDK for GameObject and MonoBehaviour workflows
that sits above the underlying transport layer. The Unity 6.0 page lists released package versions,
making the package version part of the project's compatibility decision rather than an invisible
implementation detail.
