# Source: Autodesk — FBX SDK and Animation

- **Author:** Autodesk
- **Published:** Undated
- **URL:** https://aps.autodesk.com/developer/overview/fbx-sdk
- **Retrieved:** 2026-07-11

## Summary

Autodesk presents the FBX SDK as a cross-platform C++ toolkit for applications, plug-ins, converters,
and tools using FBX technology. Its documentation covers scene import/export and animation structures,
including blended animation and takes. This ecosystem explains FBX's strength as a DCC and animation
interchange boundary, while also making version and converter behavior important to test.
