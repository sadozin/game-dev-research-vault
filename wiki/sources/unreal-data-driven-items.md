## Summary

Epic's item tutorial uses a struct for the common item schema and Data Assets/Data Tables for the
actual definitions. The data-driven arrangement keeps item content centralized, inspectable, and
scalable as the game grows, while allowing the runtime item behavior to remain separate from the data
that defines tools, consumables, names, and other properties.
