## Summary

Dungeon Defenders explicitly combines tower defense with direct action-RPG combat, character
classes, loot, cooperative play, and varied level layouts. The hybrid changes the core skill test:
success depends on both capital/coverage planning and real-time execution, positioning, gear, and
team coordination.

That combination multiplies scope. Networking, character combat, loot balance, camera/control feel,
enemy aggro, and tower simulation all become first-class production systems rather than optional
flavor.
