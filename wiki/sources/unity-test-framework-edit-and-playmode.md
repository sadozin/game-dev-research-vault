- **Author:** Unity Technologies

## Summary

Unity Test Framework supports Edit Mode and Play Mode tests, including platform runs. Unity documents
coroutine-based `UnityTest` cases for frame-dependent behavior and XML test results for command-line
execution. The separation lets a workflow select fast editor tests or runtime lifecycle tests rather
than treating all validation as one mode.
