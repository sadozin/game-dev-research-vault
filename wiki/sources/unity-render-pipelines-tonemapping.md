# Source: Unity render pipelines and post-processing documentation

- **Author:** Unity Technologies
- **Published:** continuously maintained documentation
- **URL:** https://docs.unity3d.com/Manual/render-pipelines.html
- **Retrieved:** 2026-07-11

## Summary

Unity’s Built-in, URP, and HDRP pipelines differ in feature coverage, configuration, and target
hardware. Post-processing applies screen-space effects after scene rendering; exposure and tone
mapping are part of the display transform rather than merely decorative filters. Pipeline choice is
therefore a project constraint, not a late material setting.
