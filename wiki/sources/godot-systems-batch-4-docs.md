# Source: Godot — mobile, JS bridge, TileSets, spatial shaders

- **Author:** Godot Engine documentation contributors
- **Published:** Godot 4.x stable documentation
- **Retrieved:** 2026-07-11
- **Raw:** `raw/godot-systems-batch-4.md`

## Summary

Android: OpenJDK 17, Android SDK/NDK paths in Editor Settings, adaptive icons, Play AAB +
release keystore, INTERNET permission for net. iOS: macOS+Xcode, Team ID + reverse-DNS bundle
ID, export to Xcode project, optional live folder link. JavaScriptBridge: get_interface,
create_object, create_callback (keep refs), eval, download_buffer; web feature only.
TileSets: atlas from sheets, scene collections, physics/nav/occlusion layers, custom data,
terrains (peering bits), alternatives, property paint. Spatial shaders: PBR fragment outs,
vertex displacement, uniforms, NORMAL_MAP, render modes.
