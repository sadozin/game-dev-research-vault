# Source: ink narrative scripting documentation

- **Author:** inkle
- **Published:** continuously maintained
- **URL:** https://github.com/inkle/ink
- **Retrieved:** 2026-07-11

## Summary

ink separates authored narrative text and branching logic from the runtime integration that presents
it. Knots, stitches, choices, and variables make a dialogue graph readable and serializable, while
the game remains responsible for UI, localization, saves, and gameplay side effects. The separation
reduces code coupling but does not solve branch explosion.
