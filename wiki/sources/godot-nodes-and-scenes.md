## Summary

Nodes are the smallest building blocks: name, properties, per-frame callbacks, extensible by
scripts, parent/child trees. A scene is a saved tree with one root. Projects need a **main
scene** (F5 / Run Project). Scene files live under `res://` (project root). Editor presets
(2D Scene → Node2D, 3D Scene → Node3D, User Interface → Control) are convenience only.
2D nodes end in `2D`; 3D counterparts end in `3D` (Godot 4 renamed Spatial → Node3D).
