# Source: Simon Ludgate — The F-Words of MMOs: Faucets

- **Author:** Simon Ludgate
- **Published:** 2011-07-21
- **URL:** https://www.gamedeveloper.com/design/the-f-words-of-mmos-faucets
- **Retrieved:** 2026-07-11

## Summary

Ludgate describes an MMO's money-in/money-out model: faucets inject currency through activities such
as kills and quests, while drains remove it. Because everyone shares the persistent world, uncontrolled
inflow can make it harder for new players to enter and can inflate prices. The useful warning is that
developers must inspect both sides of the flow rather than only adding rewards to improve momentary
engagement.
