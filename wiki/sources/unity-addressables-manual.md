# Source: Unity — Addressables

- **Author:** Unity Technologies

## Summary

Unity describes Addressables as an address-based asset system that resolves assets and dependencies
from local or remote locations through asynchronous loading. It presents Addressables as a managed
asset-bundle workflow that can replace direct references, Resource folders, or manually managed bundles
when a project needs dynamic loading and content location flexibility.
