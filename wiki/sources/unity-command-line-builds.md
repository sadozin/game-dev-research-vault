# Source: Unity — Build a Player from the Command Line

- **Author:** Unity Technologies
- **Published:** Undated
- **URL:** https://docs.unity3d.com/Manual/build-command-line.html
- **Retrieved:** 2026-07-11

## Summary

Unity documents command-line Player builds using a project path, quit flag, and target or build
profile. Batch mode and a dedicated log file are recommended, while custom builds use an explicit
`-executeMethod`. These arguments provide the bounded inputs an automation wrapper should validate
and record.
