# Source: MCP — Tool Approval and Execution Scope

- **Author:** Model Context Protocol project
- **Published:** 2025-06-18 specification
- **URL:** https://modelcontextprotocol.io/specification/2025-06-18/server/tools
- **Retrieved:** 2026-07-11

## Summary

MCP requires tools to be discoverable and recommends visible invocation indicators and confirmation
prompts. For build automation, the review surface should include target, output path, command/profile,
credentials or uploads, and expected side effects before execution.
