# Source: Embracing Push Forward Combat in DOOM

- **Authors:** Kurt Loudy and Jake Campbell, id Software
- **Published:** GDC 2018
- **URL:** https://www.gdcvault.com/play/1024940/Embracing-Push-Forward-
- **Retrieved:** 2026-07-14
- **Raw:** raw/doom-push-forward-combat-gdc.md (provenance only)

## Summary

id Software defines push-forward combat as an aggressive, fast power fantasy that challenges
regenerating health, reload, and cover. Instead of retreating to recover, the player takes needed
resources from adversaries. Mechanics, AI, encounters, and resource delivery therefore reinforce
continued engagement.

This is a design account of Doom (2016), not a description of classic Doom. Resource-on-kill and
close-range recovery should not be projected backward onto the 1993 attrition/exploration loop.
