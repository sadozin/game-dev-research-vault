## Summary

Unity AI Navigation provides navmesh surfaces, agents, links, and modifiers for traversable-space
queries and movement. Baking makes navigation fast at runtime, while dynamic obstacles and off-mesh
links preserve flexibility. The catch is that a valid navmesh is not a complete steering solution:
agent size, avoidance, goals, and moving-world updates still need gameplay policy.
