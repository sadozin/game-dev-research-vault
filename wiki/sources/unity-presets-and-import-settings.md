## Summary

Unity Presets save and reuse settings from asset importers, components, and project settings. The
Preset Manager can assign defaults by importer type and filter, including glob-style filters. Presets
are project assets, so storing them in version control makes import conventions reviewable and
repeatable rather than hidden in an agent's invocation.
