# Source: Unity Manual — Importing Assets

- **Author:** Unity Technologies
- **Published:** Undated
- **URL:** https://docs.unity.cn/Manual/ImportingAssets.html
- **Retrieved:** 2026-07-11

## Summary

Unity reimports assets when source files or import settings change, and exposes those settings in the
Inspector. This makes import configuration part of the asset's reproducible state and provides a
natural validation point for generated content: inspect the imported result and settings rather than
assuming the source file's appearance transfers unchanged.
