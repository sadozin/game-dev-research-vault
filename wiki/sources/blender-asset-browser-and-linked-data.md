# Source: Blender Manual — Asset Browser and Linked Data

- **Author:** Blender Foundation
- **Published:** Undated (Blender Manual)
- **URL:** https://docs.blender.org/manual/en/latest/editors/asset_browser.html
- **Retrieved:** 2026-07-11

## Summary

Blender's Asset Browser tracks the source library of linked or appended data and can reuse asset data
when an asset is used again. The manual also distinguishes making an object local from leaving mesh
and material data linked. That distinction is important when accepting generated content: the project
should know which files remain authoritative and which data has been deliberately copied or edited.
